# KreditKard

## Digitizing Nigeria's ₦3 Trillion Informal Credit Economy.

### The Problem

  ₦8 Billion Lost Daily: Paper "tally books" are destroyed, lost, or burned, causing debt records to vanish.

  Dispute Deadlocks: Customers claim they already paid with no digital proof, leading to lost relationships.

  Offline Exclusion: 70M+ Nigerians use feature phones and are excluded from smartphone-only fintech.

  Zero Infrastructure: No digital solution currently exists to manage informal credit at scale.


### The Solution

KreditKard is a digital tally system that generates instant payment options for every informal debt.

- For Traders: Record a debt in seconds and generate a unique payment code instantly.

- For Customers: Receive payment instructions via SMS and pay via USSD, transfer, or QR—no app download required.

- For the Economy: Bridges the gap between informal traders and formal digital payment rails.


### Technical Architecture

The system is built to be "Offline-First" for the customer while leveraging heavy-duty fintech APIs for the trader.

- Payment Rails: Interswitch Virtual Accounts and Flutterwave USSD API.

- Settlement: Automated routing from customer bank to trader account via Interswitch.

- Verification: Webhook-driven debt clearing to ensure real-time accuracy.

- Trust: (Phase 1) Immutable digital logs to replace paper records and prevent disputes.


### Why?

- Massive Market: Targets a ₦3 trillion addressable market with zero current digital competitors.

- API Mastery: Demonstrates deep integration of Interswitch's Virtual Accounts, USSD Billing, and Nano-Lending APIs.

- Inclusion: Works on a basic Nokia phone, removing the "smartphone barrier" to entry.

- Feasibility: Technically achievable in a 7-day sprint using existing sandbox environments.
